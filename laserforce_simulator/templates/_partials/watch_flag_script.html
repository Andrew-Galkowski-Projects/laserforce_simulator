{% comment %} LG-06f — once-per-page delegated-click handler for the watch-flag buttons. Include exactly ONCE near the end of a page's {% block content %} (NOT inside any loop). Binds a single delegated click listener that POSTs to the button's ``data-toggle-url`` with ``player_id`` from ``data-player-id``, sending the ``X-CSRFToken`` header. The token is rendered server-side via ``{{ csrf_token }}`` (which also calls ``get_token()`` so Django sets the ``csrftoken`` cookie on this response) — the league screens render GET-only forms with no ``{% templatetag openblock %} csrf_token {% templatetag closeblock %}``, so a fresh session landing directly on a league screen would otherwise have no cookie and the POST would 403. The ``csrftoken`` cookie is kept as a fallback. On ``{watched:true}`` it adds ``.watch-flag-on`` to ALL buttons sharing that ``data-player-id`` (handles Statistical Feats / League Leaders rendering a player on multiple rows); on ``{watched:false}`` it removes the class from all of them. {% endcomment %}
